# German Grammar Quest MVP

Minimal Next.js App Router scaffold for a German grammar game MVP.

## Included

- Next.js with App Router
- TypeScript
- Tailwind CSS
- Mock A1 `der / die / das` exercises
- Basic route flow:
  - `/`
  - `/avatar`
  - `/map`
  - `/lesson/a1-articles`
  - `/exercise/ex-001`
  - `/result`

## Run

```bash
npm install
npm run dev
```
